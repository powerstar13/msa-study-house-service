# ℹ️ 실행 방법

`out 폴더 경로의 각 jar 실행 명령을 입력 바랍니다.`

1. API Gateway 실행
    ```
    java -jar ./out/stattion3-assignment-gateway.jar
    ```
2. 회원 서비스 실행
    ```
    java -jar ./out/stattion3-assignment-member.jar
    ```
3. 방 서비스 실행
    ```
    java -jar ./out/stattion3-assignment-house.jar
    ```
